1. Extract the plugin to the plugins folder: adm_plugins/awards
2. Run the installation script once by visiting: adm_plugins/awards/system/awards_install.php
3. Configure the menu by adding the main plugin link:
   - Awards Main Page → /adm_plugins/awards/index.php (Icon: bi-award)
   
   The main page provides tab navigation for:
   - Manage Awards (assign/edit awards, configure settings)
   - List All Awards (with export options)
   - Jubilee Report (members with configurable anniversary years)
   
   Optional menu entries (direct links):
   - Show Awards → /adm_plugins/awards/system/awards_show.php (Icon: bi-list-ul)
   - Assign Awards → /adm_plugins/awards/system/awards_change.php (Icon: bi-award-fill)
   - Jubilee Report → /adm_plugins/awards/system/awards_jubilee.php (Icon: bi-calendar-star)
   - Edit Award Categories → /modules/categories.php?cat_type=AWA (Icon: bi-tag)
   
   Note: You must have "Administrator Users" rights to manage AWA categories.
   
4. Configure Jubilee Years:
   - Go to Awards Main Page → Manage Tab → "Configure Jubilee Years"
   - Add/remove anniversary years that should be recognized (e.g., 10, 25, 40, 50, 60, 70)
   - Default years: 25, 40, 50, 60, 70
   
5. To display awards on profile pages, run the addin installation script after initial installation
   and after each Admidio update:
   <your-admidio-installation>/adm_plugins/awards/system/awards_install_addin.php
   
   Info:
   To remove awards from profile pages, run the uninstall script:
   <your-admidio-installation>/adm_plugins/awards/system/awards_uninstall_addin.php

## Older Admidio Versions

### Admidio v4.x
For Admidio v4.x, please use Awards Plugin version 3.0.0.

### Admidio v3.3.1+
1. Extract to plugin folder adm_plugins/awards
2. Run the link adm_plugins/awards/awards_install.php once
3. Configure the menu, adding the following links:
   Show awards → /adm_plugins/awards/awards_show.php
   Assign awards → /adm_plugins/awards/awards_change.php
   Edit award categories → /adm_program/modules/categories/categories.php?type=AWA
4. To show awards on profile pages add the line:
   "include_once(ADMIDIO_PATH . FOLDER_PLUGINS .'/awards/awards_profile_addin.php');"
   at the end of adm_program/modules/profile/profile.php BEFORE $page->show();

### Admidio v3.3.0
WARNING: Category editing does not work due to a bug in Admidio v3.3.0.
Please use awards only with older or newer versions!

### Admidio before v3.3.0
1. Extract to plugin folder /awards
2. Add to Theme/my_bottom:
   include(SERVER_PATH.'/adm_plugins/awards/awards.php');
3. Open the page and click Install in the Awards section of the sidebar
