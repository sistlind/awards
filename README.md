# Awards Plugin for Admidio

Awards/Honors/Certificates Plugin for [admidio.org](https://www.admidio.org/)

## Compatibility

| Plugin Version | Admidio Version | PHP Version |
|----------------|-----------------|-------------|
| 4.0.0+         | v5.03+          | 8.2+        |
| 3.0.0          | v4.3.3          | 7.4+        |
| 2.0.0          | v4.1            | 7.4+        |

## Features

- Assign awards/honors/certifications to members
- Organize awards in categories
- Display awards on member profiles
- Export awards list to CSV, PDF
- Multi-organization support
- Batch processing (assign awards to entire roles)
